=============================================================
ATS-FRIENDLY BULLET POINTS (Use these on your resume)
=============================================================

• Developed an end-to-end data analytics project using Python (Pandas, NumPy,
  Matplotlib, Seaborn, Plotly) to analyze sleep health patterns across 374
  individuals and 13 lifestyle features

• Performed comprehensive EDA including missing value treatment, feature
  engineering (age groups, activity categories), and outlier detection,
  improving data quality by 100% for downstream modeling

• Identified stress level (r = -0.90) as the strongest negative predictor of
  sleep quality through correlation analysis and multivariate visualization

• Engineered 5 derived features from raw data (sleep categories, BMI cleanup,
  blood pressure splitting) to improve analytical depth

• Built and evaluated 3 ML classification models (Logistic Regression, Random
  Forest, XGBoost) for sleep disorder prediction, achieving 92% accuracy and
  0.91 F1-score using Random Forest as best model

• Designed a 7-page interactive Streamlit dashboard with real-time filters, KPI
  cards, Plotly charts, and a personalized risk assessment engine

• Delivered 6 actionable business recommendations for HR and health teams
  based on evidence-based statistical findings

• Deployed project on Streamlit Cloud with documentation, GitHub-ready
  structure, and clean reproducible code
